# TikChat

#### Description
高并发聊天平台的后端服务。使用SpringBoot、netty、redis、redisson消息队列、mysql、
已完成基础功能如单聊、群聊、发送图片、视频、预览视频、在线修改图片、拉黑联系人、踢群成员、解散群聊、删除好友、管理员管理、版本更新 等等。
